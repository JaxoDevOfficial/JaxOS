# JaxOS
A text-based operating system built using Python.
## Commands
<h4>help</h4>
The default and easy "help" command.
<br />
Usage: ```help [command-name]```
<br />
<br />
<h4>clear</h4>
Clears the screen. It's as easy as that.
<br />
Usage: ```clear```
